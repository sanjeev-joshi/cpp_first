# Build, tests and generate coverage reports from vscode

On workspace give <"ctrl+shift+b">
After the succesful execution, you will see build folder structure tree view like below
.
├── app
│   └── app
└── test
    ├── coverage_html
    │   ├── coverage
    │   │   └── <workspace>
    │   │           └── cpp_first
    │   │               ├── src
    │   │               │   └── logger
    │   │               │       ├── logger.cpp.html
    │   │               │       └── logger_test.cpp.html
    │   │               └── test
    │   │                   └── test.cpp.html
    │   ├── index.html
    │   └── style.css
    ├── log.txt
    ├── utest
    ├── utest.profdata
    └── utest.profraw
